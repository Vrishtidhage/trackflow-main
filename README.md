# TrackFlow

TrackFlow is a bug tracking application built with React, Vite, FastAPI, and SQLAlchemy.

## Project Structure

```text
TrackFlow-main/
|-- client/
|   |-- index.html
|   |-- package.json
|   |-- vercel.json
|   |-- vite.config.js
|   `-- src/
|       |-- api.js
|       |-- App.jsx
|       |-- main.jsx
|       |-- components/
|       |-- contexts/
|       |-- pages/
|       |-- theme/
|       `-- utils/
|-- server/
|   |-- requirements.txt
|   |-- app/
|   |   |-- main.py
|   |   |-- config.py
|   |   |-- database.py
|   |   |-- models/
|   |   |-- routers/
|   |   |-- schemas/
|   |   |-- services/
|   |   `-- utils/
|   `-- alembic/
|-- render.yaml
|-- requirements.txt
|-- start-trackflow.bat
`-- start-trackflow.ps1
```

`client/dist/` is generated by `npm run build`. Do not edit files inside `dist`.

## Local Run Commands

Install dependencies once:

```powershell
cd client
npm install
cd ..\server
.\venv\Scripts\pip.exe install -r requirements.txt
```

Run backend:

```powershell
cd server
.\venv\Scripts\python.exe -m uvicorn app.main:app --host 127.0.0.1 --port 8000
```

Run frontend:

```powershell
cd client
npm run dev -- --host 127.0.0.1 --port 5173
```

Open:

```text
Frontend: http://127.0.0.1:5173
Backend:  http://127.0.0.1:8000
```

You can also start both with:

```powershell
.\start-trackflow.bat
```

## Render Deployment

This repo includes `render.yaml` for Render Blueprint deployment.

Recommended Render services:

```text
trackflow-api     FastAPI backend
trackflow-client  React static frontend
```

Backend settings:

```text
Root Directory: server
Build Command: pip install -r requirements.txt
Start Command: uvicorn app.main:app --host 0.0.0.0 --port $PORT
```

Frontend settings:

```text
Root Directory: client
Build Command: npm install && npm run build
Publish Directory: dist
```

Required backend environment variables:

```text
DATABASE_URL=<your Render PostgreSQL internal/external database URL>
SECRET_KEY=<strong random secret>
ADMIN_EMAIL=admin@trackflow.local
ADMIN_PASSWORD=<strong admin password>
```

Required frontend environment variable:

```text
VITE_API_URL=https://your-trackflow-api.onrender.com
```

Do not use `http://127.0.0.1:8000` in production. That only works locally.

## Notes

- `server/requirements.txt` is the main backend dependency file.
- The root `requirements.txt` points to `server/requirements.txt` for platforms that look in the repository root.
- The backend CORS config allows local development, Vercel preview domains, and Render domains.
